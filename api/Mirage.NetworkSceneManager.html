    Class NetworkSceneManager

    Provides Scene Management to a NetworkServer and or NetworkClient.

    The NetworkClient loads scenes as instructed by the NetworkServer.

    The NetworkServer controls the currently active Scene and any additive Load/Unload.

    ChangeServerScene(String, SceneOperation)

    This causes the server to switch scenes and sets the ActiveScenePath.

    Clients that connect to this server will automatically switch to this scene. This automatically sets clients to be not-ready. The clients must call Ready() again to participate in the new scene.
